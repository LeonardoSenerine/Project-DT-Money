# DT Money 💰

DT Money is a learning-focused project developed during a **Rocketseat course**, aimed at practicing **React fundamentals** through the creation of a simple and intuitive **financial transactions interface**.

The application provides a clear way to visualize, organize, and manage transactions, while reinforcing core front-end concepts and best practices.

---

## 🧪 Technologies Used

- **React** – Component-based UI library
- **Vite** – Fast build tool and development server
- **JavaScript / TypeScript** (according to project setup)
- **HTML5**
- **CSS / Styled Components / CSS Modules** (adjust based on your implementation)

---

## 🎯 Project Goals

- Practice core React fundamentals
- Understand and apply component-based architecture
- Work with state management and data flow
- Build a clean and intuitive UI
- Apply concepts taught in the Rocketseat course
- Improve code organization and readability
